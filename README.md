# nanoc-less

This provides a [LESS](http://lesscss.org/) filter for [nanoc](http://nanoc.ws) using [less.rb](https://github.com/cowboyd/less.rb).

## Installation

`gem install nanoc-less`

## Usage

```ruby
filter :less
```

This filter takes no options.
